pra criar um class based view, é preciso inheritar de alguma coisa
o mais generico de todos é o generic.TemplateView
o poder do CBV é justamente poder inheritar de views que voce definiu
lembrando que é preciso mudar no file URLS a seguinte linha
from . import views
path('...', views.CLASSBASEDVIEW.as_view(), name='...'),

liste util de comandos postgres:

-pra acessar o comand line
>psql

-pra acessar uma database especifica com um user
>psql -d dataclub -U tadeup

-criar usuario:
>create user tadeup with password 'unsafe' superuser;

-listar os usuario
>\du

-listar as databases
>\l

-garantir privilegios aos usiarios
>grant all on database dataclub to tadeup;

-alterar privilegios
>alter role tadeup with createdb;
